# libft_21school

Библиотека стандартных функций С и несколько вспомогательных функций.

### Часть 1

Функции из библиотеки libc

• isalpha
• isdigit
• isalnum
• isascii
• isprint
• strlen
• memset
• bzero
• memcpy
• memmove
• strlcpy
• strlcat
• toupper
• tolower
• strchr
• strrchr
• strncmp
• memchr
• memcmp
• strnstr
• atoi
• calloc
• strdup

### Часть 2

### Работа с Makefile

```make``` - сборка библиотеки

```make clean``` - удаление объектных файлов

```make fclean``` - удаление объектных файлов и библиотеки

```make re``` - пересборка
